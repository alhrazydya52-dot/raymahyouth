🇾🇪 مجلس شباب ريمة (Youth Council of Rimah)

الموقع الرسمي لـ مجلس شباب ريمة، كيان شبابي تطوعي مستقل يهدف إلى توحيد جهود الشباب وتنمية قدراتهم للمساهمة في بناء محافظة ريمة واليمن عموماً.

تم تصميم وبناء هذا النظام ليكون متكاملاً وخفيفاً (موقع عام + لوحة تحكم إدارية) يعمل بالكامل من خلال الواجهة الأمامية بتقنيات الويب الحديثة.

✨ مميزات المشروع

تصميم عصري ومتجاوب (Responsive): يعمل بكفاءة عالية على جميع الأجهزة (شاشات الكمبيوتر، الأجهزة اللوحية، والهواتف الذكية).

لوحة تحكم مدمجة (CMS): نظام إدارة محتوى مدمج لتعديل النصوص، الرؤية، الرسالة، والأهداف دون الحاجة للبرمجة.

إدارة ديناميكية للمشاريع: إمكانية إضافة، تعديل، وحذف المشاريع والأنشطة وتصنيفها (تعليم، صحة، إغاثة، تمكين) وتظهر فوراً في الموقع.

تحديث الشعار بسهولة: إمكانية تغيير الشعار من لوحة التحكم أو ببساطة عبر استبدال صورة logo.jpg في مجلد المشروع.

أدبيات المجلس: يحتوي على الأهداف الرسمية للمجلس وشروط العضوية المستمدة من اللائحة الداخلية (المادة 5 والمادة 6).

لا يحتاج لقواعد بيانات معقدة: يعمل كملف ثابت (Static) مما يجعله مثالياً للاستضافة المجانية على GitHub Pages.

🛠️ التقنيات المستخدمة

HTML5 لهيكلة بناء الموقع.

Tailwind CSS للتصميم والتنسيق المتجاوب والسريع.

Vanilla JavaScript (جافاسكربت نقي) لبرمجة التفاعلات ولوحة التحكم.

FontAwesome للأيقونات الجمالية.

Google Fonts (خطوط كايرو وتجوال) لطباعة عربية احترافية.
